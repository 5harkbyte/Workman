Workman keyboard layout

I then came up with a set of principles to serve as guidelines to help me with the design:

  • Prioritize the home key
  • If vertical folding and reaching cannot be prevented, prioritize reaching for longer fingers and folding for shorter fingers. Place more frequent keys under stronger fingers Common bigrams should be easy to type.

A different philosophy in designing keyboard layouts
